前端自动部署+HTTPS

发表于 2018-05-29

Netlify是一个现代网站自动化系统，其JAM架构代表了现代网站的发展趋势。所谓JAM，就是指基于客户端JavaScript、可重用API和预构建Markup标记语言的三者结合。

先到Netlify选择你的仓库

Branch to deploy 默认为master 这个不用改

Biuld command 是你程序的打包命令,为hexo为例:



    Biuld command: hexo g

Publish directory是静态文件,以hexo为例:



    public

然后点DEploy site就开始部署你的仓库到netlify节点上啦！


等待2-5分钟后,部署完成！下面我们就可以绑定域名了,点击Site settings 往下拉找到Change site name修改域名


域名修改好后,就可以绑定我们自己的域名了


点击Domain Settings 找到Add custom domain 填写自己的域名,绑定自己的域名后,需要到域名服务商添加一条CNAME的解析,解析地址就是你修改的域名(注意！是修改的域名,(xxxxxxx.netlify.com)不是绑定的域名)

添加解析后需要等待半个小时左右,等解析生效

解析生效后,就可以生存SSL证书了。还是Domain settings里，找到HTTPS几个大字母,点Verify DNS configuration(如果没生效,则不能进行下一步)。带按钮变成蓝色时,再点击


Let’s Encrypt certificate,然后只需2分钟,证书就生成好了！再访问你的站点,是不是出现绿色安全字样了呢?如需强制HTTPS,只要Force HTTPS即可
