install_hexo

2018-05-26

Hexo 是高效的静态站点生成框架，她基于 Node.js。 通过 Hexo 你可以轻松地使用 Markdown 编写文章，除了 Markdown 本身的语法之外，还可以使用 Hexo 提供的 标签插件 来快速的插入特定形式的内容。


1.安装Git
2.安装Node.js
3.创建Guthub账号

Git和Node.js安装好后,就可以开始安装Hexo了,进入cdm输入:


    
        npm install -g hexo

再在桌面上新建一个Hexo的文件夹

cmd安装完成之后

在cmd中进入你创建的hexo文件目录,输入:


    
        $ hexo init

上一步执行完成后,就可以输入Hexo s就可以本地查看了,地址是localhost:4000
下面就开始配置SSH

1.打开Git Bash here终端


git config –global user.name “你的GitHub用户名”
git config –global user.email “你的GitHub注册邮箱”

2.生成ssh密钥



    ssh-keygen -t rsa -C “你的GitHub注册邮箱”

提醒输入密码直接回车即可

上面步骤完成后,在用户目录下会有一个.SHH文件，复制id_rsa.pub里的内容

在GitHub里 New SSH Key,title随意,key就填id_rsa.pub里的公匙

测试SSH


    
        ssh -T git@github.com

接下来会是类似这样的提示内容:


    
The authenticity of host ‘github.com (0.0.0.0)’ can’t be established.
RSA key fingerprint is 17:24:ab:cd:ef:28:24:bf:62:1b:tg:4d:eb:gd:a6:45.
 Are you sure you want to continue connecting (yes/no)?

输入Yes


到这里就已经部署完成了,如果需要绑定域名,到GitHub绑定就可以看到效果了！
